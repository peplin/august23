About

This code drives the Wiremap when I perform it. It's not designed to be universally user friendly, but more Albert-friendly when I'm on stage.

Instructions

When no keys are pressed, the mouse will control the X and Z of the globe.

  • Press "1" (one) to make the mouse control Y and Z
  • Press "2" to make the mouse control X and Z
  • Press "3" to make the mouse control X and Y
  • "w" and "s" will make the globe travel along the axis the mouse can't (for "1", "w" and "s" control the X axis)
  • "e" and "d" control the radius of the globe
  • Holding down "q" freezes the globe at it's current location
  • Holding down "a" makes the mouse control the radius of the globe
  • Pressing "z", "x", "c" will set the rgb value of the globe in relation to mouse location
  • Pressing "v' will set the grayscale of the globe

There is also a beat mapper. To use it, record the first beat of a measure by hitting "t", and the first beat of the next measure by hitting "y". Then:

  • "u" turns on the pulse
  • "i" turns the pulse off
  • "o" turns on the strobe
  • "p" turns the strobe off
  • "7" turns on the disco ball
  • "8" turns it off
  • "g", "h", "j", and "k" modify the tempo - whole, half, quarter, and eight

This is all built with Processing, and here is the source code.
